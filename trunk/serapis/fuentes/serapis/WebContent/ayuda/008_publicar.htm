Publicar

Esta opción permite publicar los documentos o “subir” los documentos desde el equipo local a SERAPIS, para lograr esto SERAPIS presenta una página con un formulario que permite publicar el documento, este formulario esta compuesto por:

  • Proceso. la selección del proceso al cual pertenece el documento que se subirá.
  • Tipo de Documento. Establece el tipo de documento que se subirá.
  • Nombre. Permite la selección del nombre del documento, los nombres de los documentos aparecerán dependiendo de la selección de las opciones realizadas para Proceso y Tipo de Documento.
  • Versión Plantilla. Permite que se ingrese la versión del documento, este es un número que va desde 01 a 99. Siempre se deben ingresar los dos dígitos.
  • Descripción Es un comentario que se puede agregar al documento y que sirve de descripción para este.
  • Archivo Es la ubicación del archivo. Con el botón Examinar se puede ubicar el archivo que se subirá a SERAPIS.
 

Una vez que se ingresan todos los campos anteriores, por medio del botón Subir se agregará el documento al Sistema, al subir se realizarán una serie de validaciones, si estas se cumplen en su totalidad el documento estará disponible. Si existiese un documento que corresponde al Proceso, Tipo de documento, Nombre y Versión Plantilla, se avisará que el documento no es posible subirlo ya que existe uno previo con las mismas características.
Una vez que el archivo ya se cargue, se mostrará una página especificando que el archivo se copio directamente al Sistema.

 
Asignación de WorkFlow


Todos los documentos que es posible que maneje SERAPIS están previamente definidos, uno de los componentes de la definición corresponde a un WorkFlow, el que permite realizar la autorización del documento en “n” pasos donde existen diferentes usuarios involucrados. Si el documento que se publica tiene asociado un WorkFlow el documento queda en estado En Flujo lo que significa que el documento esta en espera de aprobación, cuando este es el caso, SERAPIS informa que el archivo se copio exitosamente al sistema, pero que ha ingresado a un flujo de aprobación.
