Configuración - Tipo Documento


Esta opción permite agregar, editar o eliminar un tipo de documento. Los tipos de documentos se identifican con un código que corresponde a una letra mayúscula y son establecidos por el administrador.
Para editar o eliminar un tipo de documento haga clic en el código del tipo de documento.
Para agregar un tipo de documento presione el botón Nuevo.
Para imprimir el listado presione el botón Imprimir.

 

Al momento de editar un tipo de documento sólo se puede modificar la descripción de éste. Sin embargo es posible eliminar el tipo seleccionado en esta misma pantalla.

 
 

La creación de un tipo de documento nuevo (Figura 86) requiere la definición de:
Código. Corresponde a un solo carácter de extensión. Descripción. Permite tener una referencia rápida del tipo.
