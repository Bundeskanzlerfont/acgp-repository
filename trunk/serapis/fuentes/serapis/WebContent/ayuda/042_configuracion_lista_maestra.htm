Configuración - Lista Maestra


Esta opción define la lista maestra de documentos que utiliza el sistema. Los documentos se encuentran divididos por Proceso y la información se muestra según: Tipo Documento, Nombre, Aprobación y Ejecución (Workflow) y Tiempo Retención.

 

Para editar un documento se debe hacer click en el icono del documento. La ventana que aparece en la imagen anterior muestra distintos campos editables, sólo Proceso y Tipo de Documento no tienen la opción modificable.
Para guardar los cambios, debe presionar el botón Grabar. Puede también borrar un documento presionando el botón Eliminar.
Para crear un documento presione el botón Nuevo ubicado al final de la lista. Deberá ingresar la siguiente información:

  • Tipo Documento. Debe seleccionar una de las opciones que presenta el listado.
  • Proceso. Permite seleccionar el proceso al cual pertenece el documento.
  • Nombre. Corresponde a una breve descripción del documento con un máximo de 50 caracteres.
  • Workflow Aprobación/Ejecución. Es posible asociar al documento un workflow de aprobación y/o de ejecución. Esto permite que cada vez que se realice una publicación del documento (ya sea por gestión de documentos u operación) se inicie un flujo de aprobación para que este quede finalmente vigente.
  • Lista Distribución. Este campo permite asociar al documento una lista de distribución previamente creada.
  • Tiempo Retención. Tiempo que debería permanecer el documento en el sistema.
