Configuración - Gestión de Procesos

Este mantenimiento es utilizado para especificar cual es el registro de Gráfico de gestión, Minuta reunión comité y los registros de auditoria, estos son utilizados para entregar el listado de registros exactos para la gestión de procesos.

 

Para hacer esto, SERAPIS lista los registros para que se seleccione el correspondiente a los gráficos de gestión y Minuta reunión comité. Para el caso de los registros de auditoria como pueden ser varios, en la lista de la izquierda que se observa en la Figura 91 esta el listado de registros, para seleccionar uno se debe marcar y presionar sobre el botón >>, para eliminar un registro de la lista, seleccione el elemento de la lista de la derecha y presione sobre <<.
